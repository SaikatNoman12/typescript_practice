# TypeScript

### Install typescript in your device

    npm i -g typescript

### Type script run command

    // 1st:

    tsc script.ts

    // 2nd:

    tsc --init

    rootDir:'./src';  // go tsconfig.json

    outDir:'./output';  // go tsconfig.json

    tsc   

    // or

    tsc -w  // always run typescript file.
